Overview

Purpose

This is a portfolio to showcase my skills, projects and experience for future employers. I will be showcasing my past projects that I worked on in a nice display for this website.

Audience

It's for internships and jobs that I plan to apply for, because when applying for jobs it gives one an upper hand to have a portfolio on hand. I also want to add this portfolio on my linkden profile.

Dynamic elements

The main difference between my previous portfolio and this one is that I'll be adding js as a search bar. There will be a search bar to find projects depending on the inputed keywords, so employeers who want to see my projects that incorporated js can see my related projects.
